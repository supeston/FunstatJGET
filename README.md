# FunstatJGET

бот для автоматизации записи на квесты jget, ведения личной статистики и просмотра топов всяких.

сделал его, потому что надоело руками сидеть по субботам и обновлять сайт, чтобы успеть забрать нормальные смены до того как их расхватают за секунды.

что бот умеет:
- автозапись и перехват: бот сам пишет вас по субботам по выбранным фильтрам (можно настроить школы, типы квестов, станции, время и сколько смен макс нужно). еще есть фоновый перехват который сам чекает сайт каждые 30 сек в будни и если кто-то отменил смену - бот ее тут же ловит.
- профиль со статистикой: все данные в одном месте. бот показывает сколько всего смен было проведено за все время, сколько отмен, опозданий и общее отработанное время в часах. 1 смена = 1 час, все четко считается.
- деньги: считает точный заработок за все время, сколько заработали, сколько еще в ожидании висит.
- план на день: быстро глянуть расписание на сегодня или завтра со школой и станцией, есть сразу линк на карты чтобы маршрут построить.
- библиотеки: показывает сколько часов вы отработали по конкретным школам, квестам и отдельным станциям.
- топы: можно чекать глобальные топы по часам, по бабкам, по всем сменам вообще, по опозданиям и по отдельным квестам (дружба, пдд и так далее). данные берутся за все время.
- осинт (пробив по базе): можно вбить любого чела по имени или фамилии и посмотреть его полное досье, общую стату, бабки и историю последних смен, если он вообще есть в базе бота.

как запустить:
1. вставить токен бота в файл .env
2. запустить main.py
3. в боте уже привязать свой аккаунт с сайта jget
4. настроить фильтры для записи под себя и кайфовать
