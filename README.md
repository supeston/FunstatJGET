# FunstatJGET

Бот для автоматизации записи на квесты в J-GET, ведения личной статистики и просмотра топов.

Я сделал этого бота, потому что самому надоело постоянно сидеть и обновлять сайт по субботам, чтобы успеть урвать нормальные смены до того, как их разберут за секунды.

Что сейчас умеет делать бот:

* **Автозапись и перехват смен**: Бот автоматически записывает вас по субботам (для второгодников в 10:00, для первогодников в 12:00) по вашим фильтрам (настройка по конкретным школам, типам квестов, номерам станций, времени и лимиту смен). Также работает фоновый перехват, который каждые 30 секунд ищет свободные смены в будни и бронирует их.
* **Удобный профиль**: Вся статистика собрана в одном месте. Бот показывает общее число проведенных смен, отмен, опозданий и общее количество отработанных часов.
* **Финансы**: Показывает точные цифры заработка за все время (уже заработано, смены в ожидании и суммарный доход).
* **План дня**: Быстрый просмотр расписания на сегодня или завтра (время, школа, категория квеста, станция/роль) с быстрой ссылкой на Яндекс Карты для построения маршрута.
* **Библиотеки часов**: Показывает, сколько часов вы отработали по конкретным школам, квестам и станциям.
* **Глобальные топы**: Можно соревноваться и смотреть, кто на каком месте по часам, заработку в рублях, общему числу смен, опозданиям, прогулам и отдельным типам квестов. Листать топы можно прямо кнопками-стрелочками под меню.
* **ОСИНТ-Поиск (для админа)**: Поиск любого человека в базе данных смен по его имени или фамилии, чтобы посмотреть его полное досье, общую статистику, финансы и историю последних смен.

Настройки и запуск:
1. Прописать токен бота в файле `.env`.
2. Запустить `main.py`.
3. Привязать свой аккаунт сайта jget-events.ru в меню бота.
4. Настроить фильтры для автозаписи под себя.
