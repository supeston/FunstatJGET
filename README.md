# FunstatJGET 🤖🛸

Всем привет! 👋

Я создал этого бота, потому что сам столкнулся с проблемой постоянной записи на квесты в системе J-GET. Это было жутко неудобно — постоянно караулить время открытия записи по субботам, бесконечно обновлять страницу в браузере и надеяться, что успеешь занять нужные слоты до того, как они разлетятся за секунды.

В какой-то момент мне надоело заниматься этим вручную, и в голову пришла простая мысль: *«А почему бы не автоматизировать весь этот хаос?»* 💡

Так и родился этот проект. Он создавался для удобства и экономии времени. Бот помогает решать самые рутинные задачи в один клик:
* **Авто-штурм**: Высокоскоростное автоматическое бронирование мест для 50+ ботов параллельно по субботам ровно в 12:00 по МСК (с отложенным 14-минутным кулдауном для отмены).
* **Комбо-маршруты**: Умная группировка квестов в цепочки для получения максимального дохода за день.
* **Аналитика и ОСИНТ**: Отслеживание выплат ведущим и главарям, рейтинги, графики заработка и поиск по базе.
* **Мониторинг сети**: Проверка пинга до API и актуального статуса сессий ботов прямо из админ-панели.

Этот бот — мой личный вклад в упрощение жизни всех, кто работает с J-GET. Всё сделано интуитивно, быстро и надёжно. Надеюсь, проект окажется для вас полезным! 🚀
